#### 👨‍💻 About Me:

- `SprintBoot` 와 `Java` 를 사용해 효율적 / 체계적인 프로그래밍을 추구하고 설계합니다.
- 지속적인 유지보수와 시스템 최적화를 고려하여 프로그래밍 합니다.
    - **비동기 처리**를 통해 API 응답 속도를 `4.141s` 개선한 경험이 있습니다.
    - **캐싱**을 통해 간접 동시성 제어 및 API 응답속도를 `74ms` 개선한 경험이 있습니다.
    - **동기화**를 통해 스레드 간 경쟁 조건/ 데이터 일관성 문제를 방지한 경험이 있습니다.
    - **CI/CD**를 구현하여 개발/배포 프로세스 효율을 높인 경험이 있습니다.
- 유창한 영어(🇺🇸), 포르투갈어(🇵🇹🇧🇷)구사가 가능하며 글로벌 협업이 가능합니다
